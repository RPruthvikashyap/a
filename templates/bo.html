002 : Budget Organiser

"This project helps us managing day to day transactions and helps us assist better financial planning the in future. It also keeps our record organized with the benefit of never forgetting where we have spent our expenses."

Github Link

Github

Objectives of the Project

  • The software provides individual accounts to ensure privacy and security.
  • It integrates different modules within the system.
  • Offers a simple entry process for every transaction.
  • Allows users to update and delete modules.
  • Provides detailed reports of overall transactions.

Module Descriptions

  • Register: New users can register using email, username, opening balance, and password.
  • Login: Users can log in using their email ID and password, leading them to the homepage.
  • User Account Management: Users can update, create, or delete their account.
  • Income and Expense: Users can enter income and expense transactions and upload receipts.
  • Credit and Credit Repayment: Users can enter credit amounts and upload receipts.
  • Savings and General Reserve: Users can enter savings amounts and upload receipts.
  • Logs: Keeps a record of all entries made.
  • Taxation: The taxation module provides dynamically updated tax amounts per fiscal year.

Scope of the Project

  • The software ensures that the money spent and earned is systematically maintained.
  • The project helps manage day-to-day transactions and prevents users from forgetting where expenses have been made.

Technologies Used

  • Front End: Bootstrap, HTML, CSS, and JavaScript are used for creating the user interface and ensuring a responsive design.
  • Back End: Python is used for server-side logic and application development, with Django as the web framework.
  • Database: SQLite is used as the database for storing and managing data, providing a lightweight and efficient solution.
  • System Compatibility: The system can be accessed on any device that has an up-to-date web browser.
